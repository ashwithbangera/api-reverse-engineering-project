#  API Reverse Engineering Project

## 📌 Overview

This project demonstrates how to reverse engineer APIs from a web application without official documentation using mitmproxy and Postman.

---

## ⚙️ Tools Used

* mitmproxy
* mitmproxy2swagger
* Postman
* Docker
* Kali Linux

---

## 🚀 Steps Performed

1. Captured traffic using mitmproxy
2. Saved flows file
3. Converted flows to Swagger
4. Extracted API endpoints
5. Imported into Postman
6. Tested APIs

---


## 🎯 Key Learning

* API communication understanding
* Reverse engineering techniques
* API testing using Postman
